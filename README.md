# Interactive Portfolio

## Overview
- Personal portfolio with contact info, my skills and projects
- Includes interactive 3D cube built with Three.js and vanilla JS

## Feature
- Responsive layout eith sections: Hero, About, Skills, Projects, Contact
- Navigation bar that scrolls to the content
- Click to change colour 3D cube
- real-time links to github

## Tech
- HTML 
- CSS3
- vanilla JavaScript
- Three.js

## How to run
1. download repository
2. open in VS code
3. open index.html in a browser ( or use VS code Live Server)
